# SDVX-editor
SDVXの低難易度譜面データを作成するために作った譜面エディタです

# 作成経緯
SDVX譜面保管所には低難易度の譜面画像がなかったため、低難易度譜面を自力で作成する必要があったからです。

# 使い方
1. "Download ZIP"や "git clone"でコードをダウンロード

2. `python score_editor.py`で実行

3. "BT"と最初に表示されているプルダウンからノーツの種類を選択できる

4. "16分"と最初に表示されているプルダウンから12分 or 16分どちらでノーツを配置するか選択できる

5. エディタ上を一度クリックしたらノーツを配置できる

6. 配置してあるノーツをクリックすると消せる

7. 一番下の部分で保存するファイル名を入力し、「jsonに出力して保存」でjson形式で譜面データを保存することができる

8. 一番上の部分の「ファイル選択」ボタンから以前作成したjson形式の譜面データを選択し、「読み込む」ボタンを押すと、以前作成した譜面データを読み込むことができる

# 注意点
・つまみは左右どちらに回すかで入力します

・低難易度譜面専用のため、24分以上の間隔や一般的でない音符間隔には対応していません

・4分の4拍子にしか対応していません

・研究で使用する譜面データ形式とは異なるので、変換するツールが別途必要です(未完成)
