# SDVX-editor
SDVXの低難易度譜面データを作成するために作った譜面エディタです

<img width="328" height="467" alt="Image" src="https://github.com/user-attachments/assets/9536d547-413f-4c75-8713-00fbd64addfc" />

16分間隔、12分間隔の2通りでノーツを配置できます

つまみは左右どちらに回すかで入力します

低難易度譜面専用のため、24分以上の間隔や一般的でない音符間隔には対応していません

また、4分の4拍子にしか対応していません

json形式で譜面データを保存することができます　また以前作成した譜面データを読み込むこともできます

研究で使用する譜面データ形式とは異なるので、変換するツールが別途必要(未完成)
